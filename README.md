# AdoNetCRUDSample

## Description

### This example demonstrates how to connect to a database and perform CRUD operations using the C# language.
- Create database MyFriends if not exists
- Create table Friends if not exists
- Add fields to table Friends
- (C)reate: Add a new records to the Friends table.
- (R)ead: Retrieve records from the Friends table.
- (U)pdate: Update an existing records in the Friends table.
- (D)elete: Delete a records from the Friends table.
